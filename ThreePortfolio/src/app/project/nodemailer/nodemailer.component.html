Email Delivery Server

- Using NodeJs and Express

Introduction

Every portfolio is designed with the sole purpose that it should encourage future job and business prospects. While your design should be convincing, the Client or Recruiter must be provided with a way to contact you. This Mailer server solved that problem. It uses the following dependencies:

  • Nodemailer
  • Nodemon
  • ExpressJs
  • Mongoose

    • API Documentation

    Send Email API

    This POST API takes a JSON input and sends mail to my email account using Nodemailer. After successful operation of Sending Mail, the emailID & UserName is stored in the MongoDB Atlas Database if the Entry is Unique.

    The Above Code snippet is an example of Form Data Provided to the API as an Input which is linked to my Contact Me Form.

    Get Statistics API

    This GET API which returns the DATA of the Mail Server as a JSON object. This Data is provided by sending a GET request to the below end-point.
    url = "https://parth-kathuria-nodemailer.herokuapp.com/api/getEmailStats"

    The Endpoint returns total Number of Emails sent using the Send Mail API and the Number of Unique Users that have used the API service.

    This API is used in the Home Page in Stats For Nerds Section to display stats in form of progress spinner.

    Closing Remarks

    This is a very basic API which uses Express and Nodejs to send email and talk to backend. It also has a few shortcoming that are:

  • It does have any user based authentication, making it a target for cyber attacks
  • User Data is not hashed before storing

    • These functionalities will be implemented in Future Updates.
