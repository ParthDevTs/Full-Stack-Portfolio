Introduction

This Portfolio Project is an extensive insight to my Web Development career. The site summarizes my journey thus far in my career along with the technologies i have used along the way to get me here.

This site has been optimised to look well on a range of devices such as tablets, mobiles and PCs. The emphaisis was to provide a rich experience along with a consistent theme which is followed throughout the website.

Technologies Used

As this is a FullStack Website, it uses a range of technologies which are used to develop Frontend, backend and Database.
The Technologies are listed below:

  • ANGULAR

    Angular is one of the most popular front end frameworks that is available to use which offers a hassle free experience along with many useful in-built features such as a Router, Angular Animations , Pipes, RxJS support, Change Detection Ref, two way binding , Light Dependency Injection .

    This makes making single page applications very easy.

  • NODE JS

    NodeJs is a javascript asynchronous event-driven runtime environment which is used to build scalable applications.

    Here, I am using NodeJs in Tandem with ExpressJs , another Javascript framework which is a major part of M.E.A.N stack applications. To connect with backend, I have used Mongoose which can also be used to create schema.

  • MongoDB Atlas

    MongoDB Atlas is a Cloud based database service which is built and managed by MongoDB. It provides users cloud data base which can be deployed at an instant without having to use on-premise database

    It is a NoSQL database and with NodeJs it is very easy to setup and use. This, along with Angular , NodeJs and ExpressJs, completes M.E.A.N stack.


These technologies coupled with some libraries like PrismJs, ParticleJs, AnimeJs and TypewriterJs complete this Portfolio website.

Features
