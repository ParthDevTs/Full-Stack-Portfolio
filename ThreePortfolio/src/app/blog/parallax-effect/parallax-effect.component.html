Parallax Effect


October 19th, 2021


Introduction

Parallax scrolling is a web site trend where the background content (i.e. an image) is moved at a different speed than the foreground content while scrolling.

For Reference, you can see the image above stays put while content moves.

Preparing The Code

While using third party libraries might be more appealing, when yur goal is to reduce the bundle size, the plan old CSS (or SCSS) can get the job done with only a few lines of code.

in this effect, you will target a container and change its background properties so that the "Parallax" effect takes place.

Here, .hero is the target class in which you want to add a parallax effect.

Add the image background and background color by background: #444; and background-image: url("/assets/anyImage.jpg");, now add background-blend-mode: multiply; to make our text more readable on screen by darkening the image.

Lastly, add background-size: cover; make it stretch to the background size. Your image background is now prepped for transformation to parallax effect.



CSS/SCSS Code


.hero{
    background: rgb(95, 95, 95);
    background-blend-mode: multiply;
    background-image: url("/assets/code1jpg.jpg");
    background-size: cover;
}


HTML Code


<section class="hero">
    <p class="display-1 text-center">Parallax Effect</p>
</section>




Adding parallax Effect

Just add background-attachment: fixed; to the above prepped code and BOOM! Parallax effect!



HTML Code


.hero{
    background: rgb(95, 95, 95);
    background-attachment: fixed;
    background-blend-mode: multiply;
    background-image: url("/assets/code1jpg.jpg");
    background-size: cover;
}
There you have it! Now add parallax effect and impress everyone. Happy Coding!
