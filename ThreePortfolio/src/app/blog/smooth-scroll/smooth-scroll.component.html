Smooth Scrolling


October 24th, 2021


Introduction

Smooth scrolling is an event driven effect in which after some interaction from User , the website scrolls itself to a set position.

For Reference, you can click on buttons below to try it:

Preparing The Code

Again, the goal of this blog is to make the use of Third Party Libraries as low as possible. So, we will explore Angular's inbuilt function i.e ViewportScroller

You'll need a container to scroll to and give it some height and an id. Here we will give: <section id="section1"> </section>

Here, id="section1" is the target id to which you will scroll to when an event gets triggered

Add the some height and background color by background: #444; and height: 100vh;

.

HTML Code


  <section class="section1" id="section1">
      <!-- some content -->
      <!-- some content -->  
      <!-- some content -->
  </section>


CSS/SCSS Code


  .section1{
      background: #444;
      height: 100vh;
  }





Adding Smooth Scroll Effect

Now, go to your component.ts file and inject a inbuilt ViewportScroller to you constructor like:
constructor(private scroller: ViewportScroller){ }

Create a function scrollToSection1(){ }, it will be responsible for scrolling to certain element when certain event calls it. Here we will use Click event on the button to call this function.



Scroll Function Defination


  constructor(private scroller: ViewportScroller){ }  
  
  
  scrollToSection1(){
    this.scroller.scrollToAnchor('section1');
  }


Lastly, Linking the button to the above defined funtion using ng-click directive of angular so that whenever user clicks this button, the function gets called.

HTML Code

    
    <button (click)="scrollToSection1()"> Scroll To Section1 </button>
    
    <section class="section1" id="section1">
      <!-- some content -->
      <!-- some content -->  
      <!-- some content -->
    </section>
There you have it! Now add Smooth Scroll effect and impress everyone. Happy Coding!
